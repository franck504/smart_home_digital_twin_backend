# Smart House 3D

Smart House 3D est une application interactive permettant de visualiser et de gérer une maison intelligente en 3D. Ce projet utilise React et Vite pour une expérience utilisateur rapide et fluide.

## Prérequis

Avant de commencer, assurez-vous d'avoir les outils suivants installés sur votre machine :

- **Node.js** (version 16 ou supérieure) : [Télécharger Node.js](https://nodejs.org/)
- **npm** (inclus avec Node.js) ou **yarn** pour gérer les dépendances.
- **Git** pour cloner le dépôt : [Télécharger Git](https://git-scm.com/)

## Installation et démarrage

1. Clonez le dépôt :
   ```bash
   git clone <URL_DU_DEPOT>
   cd smart-house-3d
   ```

2. Installez les dépendances :
   ```bash
   npm install
   ```

3. Démarrez le serveur de développement :
   ```bash
   npm run dev
   ```

4. Ouvrez votre navigateur à l'adresse [http://localhost:5173](http://localhost:5173).

## Fonctionnalités

- Visualisation 3D de la maison intelligente.
- Contrôle des appareils connectés en temps réel.
- Interface utilisateur moderne et réactive.
- Intégration avec des services IoT.

## Technologies utilisées

- **React** : pour la création de l'interface utilisateur.
- **Vite** : pour un développement rapide avec HMR (Hot Module Replacement).
- **Three.js** : pour le rendu 3D.
- **ESLint** : pour maintenir un code propre et cohérent.
