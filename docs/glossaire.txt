FastAPI : Framework moderne et performant basé sur Python pour la création rapide d'interfaces de programmation (API).
Flutter : Kit de développement logiciel (SDK) créé par Google pour concevoir des applications multiplateformes à partir d'une base de code unique.

InfluxDB : Base de données de séries temporelles conçue pour stocker et traiter de grands volumes de données horodatées provenant de capteurs.
Jumeau Numérique (Digital Twin) : Réplique virtuelle dynamique d'un système physique, synchronisée en temps réel grâce à des flux de données IoT.
MQTT : Protocole de messagerie léger et efficace basé sur le modèle publication/abonnement, optimisé pour les réseaux à faible bande passante.
MicroPython : Implémentation allégée du langage Python 3 optimisée pour s'exécuter sur des microcontrôleurs comme l'ESP32.
Smart Building : Concept de bâtiment intelligent utilisant des technologies connectées pour améliorer l'efficacité énergétique, la sécurité et le confort des occupants.
Three.js : Bibliothèque JavaScript permettant de créer et d'afficher des scènes 3D interactives dans un navigateur web.
WebSocket : Protocole de communication offrant un canal bidirectionnel et permanent entre un client et un serveur pour des mises à jour en temps réel.
Wokwi : Environnement de simulation électronique permettant de tester des circuits et du code embarqué sans matériel physique.
